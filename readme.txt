<h1>ContactApp</h1>
Learning Javascript by creating a contact App

Features:

adding and deleting Contatcs to a data structure (no database)
showing contact when clicked on corresponding item in the contact list
edit, save and update allready created contacts
Possible ways to expand project:

highlight active contact in list
connect to database or firbase
add image uploading
add dropdown to filter if contact is work related or a friend
sort contact-list alphabetically
